# Style Transfer Monorepo

## Apps

- `apps/frontend`: React + Vite landing/studio UI
- `apps/backend`: Fastify AI backend (WebGPU + Hugging Face Transformers)

## Run landing page

```bash
npm run landing
```

## Run backend

```bash
npm run dev:backend
```
